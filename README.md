# 🌐 Akarsh's Portfolio Website

A modern, responsive, and performance-optimized personal portfolio website built to showcase my projects, skills, and journey as a developer.

🔗 **Live Website:** https://portfolio-akarsh-delta.vercel.app  
📂 **Repository:** https://github.com/NotJustAkarsh/portfolio-akarsh  

---

## 🚀 Overview

This portfolio is designed to represent my identity as a developer and creator. It highlights my work, technical skills, and provides a seamless way for visitors to connect with me.

The project focuses on:
- Clean UI/UX
- Fast performance
- Responsive design
- Scalable structure

## ✨ Features

- 🎯 Minimal and modern UI design  
- 📱 Fully responsive (mobile + desktop)  
- ⚡ Fast loading with optimized assets

## 🛠️ Tech Stack

### Frontend
- React.js – Component-based UI development  
- Next.js – Server-side rendering & routing  
- Tailwind CSS – Utility-first styling  

### Deployment
- Vercel – Hosting and CI/CD  

### Tools & Libraries
- Framer Motion (for animations)  
- React Icons  
- Custom reusable components

## ⚙️ Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/NotJustAkarsh/portfolio-akarsh.git

# Navigate to the project folder
cd portfolio-akarsh

# Install dependencies
npm install

# Start development server
npm run dev
```

Open http://localhost:3000 in your browser.
